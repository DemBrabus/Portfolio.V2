EMC

Essential Macro Calculator is simple web application that aims to make calculating macros for athletes and average people easier and more timely. I personally would have benefited from a web app like this when I was trying to get lean. At the time I couldn't find a tool like this so I decided to build it with the intention of helping others in their journey to living healthier.

Website Github

Mobile First Develpment

When I designed this app, I anticipated the majority of users would be accessing this app on their mobile devices but I also wanted to have a quality experience for users interfacing with the app on a desktop or laptop.

More is not always better

When deciding on how to go about building this Web App, I thought of many options but I always came back to simplicity. Due to the features of the App I decided to use Vanilla JavaScript for its main functionality and the GreenSock library for the intro splash screen. In the future if I decide to add on to this Application I could introduce a bundler like Parcel but as of now it seemed a little over kill.
